# Vanish (RU)

Простой плагин Vanish для Paper 1.21.x:
- команды `/v` и `/v <игрок>`
- фейковые сообщения выхода/входа при переключении ваниша
- плейсхолдеры PlaceholderAPI для онлайна
- совместимость с Chatty (скрытие join/quit для vanished)
- корректный онлайн в списке серверов (vanished не учитываются)

## Требования

- Paper или Purpur `1.21.x`
- Java `17+`

Опционально:
- PlaceholderAPI (для `%vanish_online%`, `%vanish_vanished%`)
- LuckPerms (для удобной выдачи прав)
- Chatty (совместимость уже встроена)

## Команды

- `/v` - включить/выключить vanish
- `/v <игрок>` - включить vanish и телепортироваться к игроку
- `/vanishreload` (алиас: `/vr`) - перезагрузить конфиг

## Права

- `vanish.use` - использовать `/v`
- `vanish.see` - видеть vanished-игроков
- `vanish.reload` - использовать `/vanishreload`

## Плейсхолдеры

- `%vanish_online%` - онлайн минус vanished онлайн-игроки
- `%vanish_vanished%` - количество vanished онлайн-игроков

## Возможности

- Игрок в ванише скрывается в TAB и для обычных игроков.
- В списке серверов (мультиплеер-меню) vanished скрываются из онлайна.
- Состояние vanish сохраняется после переподключения.
- Реальные join/quit сообщения для vanished скрываются.
- При повторном входе vanished-игрок получает уведомление, что он всё ещё невидим.
- Поддержка hex-цветов в конфиге: `&#RRGGBB`.
