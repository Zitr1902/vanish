# Changelog

## 1.0.0

- Initial release.
- Added vanish toggle command: `/v`, `/vanish`.
- Added target teleport mode: `/v <player>`.
- Added config reload command: `/vanishreload` (`/vr`).
- Added LuckPerms-friendly permissions.
- Added PlaceholderAPI support:
  - `%vanish_online%`
  - `%vanish_vanished%`
- Added fake quit/join broadcast on vanish toggle.
- Added hex color support in messages (`&#RRGGBB`).
- Added persisted vanish state on reconnect.
- Added Chatty compatibility using `vanished` metadata.
- Fixed server list online counter and sample visibility for vanished players.
